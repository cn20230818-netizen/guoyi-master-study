# 隐私政策草案

最后更新：2026-04-14

## 定位

本网站当前定位为中医文献学习与辨证教学辅助工具，不直接向患者提供自动诊断或自动处方服务。

## 当前版本收集什么

当前版本默认不收集以下数据：

- 不创建用户账号
- 不上传用户输入的病情描述
- 不采集联系人、照片、定位、麦克风、相册等敏感权限
- 不集成广告 SDK
- 不做个性化追踪

## 当前版本如何处理数据

- 所有交互内容默认只在设备本地运行
- 学习结果基于内置规则与公开文献映射生成
- 公开医案中的方药、克数与服法仅作学习材料展示

## 未来如果接入云端能力

若后续版本增加以下能力，必须同步更新本政策并完成商店重新申报：

- 用户登录
- 云端病例保存
- 分析统计或崩溃监控 SDK
- 第三方模型或远程接口
- 消息推送

## 医疗边界

- 本网站不替代线下问诊、急诊评估和执业医师处方决策
- 若出现急性偏瘫、意识障碍、喷射性呕吐、高热抽搐等情况，应立即就医
- 任何学习结果都不能作为真实患者的直接用药依据
