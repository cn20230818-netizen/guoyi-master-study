# 发布清单

## 本地已完成

- 已创建 Expo / React Native 工程
- 已完成手机端首页、大师切换、五步辨证、学习结果与文献线索
- 已在应用内加入急重分流、隐私摘要和使用边界
- 已补充 `app.json`、`eas.json`、隐私政策草案、商店描述草案

## 真正提交前必须完成

### Apple

- 安装完整 Xcode
- 登录 Apple Developer 账号
- 在 App Store Connect 创建 App
- 配置 Bundle ID、签名证书和 Profile
- 提供支持 URL、隐私政策 URL、截图和审核备注

### Google Play

- 登录 Google Play Console
- 创建应用并确认包名
- 填写应用内容、数据安全、健康类相关声明
- 上传隐私政策 URL、图标、截图和功能说明
- 准备 AAB 包并发起审核

## 医疗相关风险点

- 不建议把产品描述写成自动诊断或自动开方
- 所有学习结果都应保留“教学用途”提示
- 若后续加入用户病例上传、云端存储或模型推理，需重新评估隐私与合规要求
